# team-maker
いろんな場面でのチーム分け支援

きっかけ: https://qiita.com/koichi2426/items/0a0896dc0f3a62e91d18

必要ライブラリ:
- pulp
- networkx（今は使ってないけど多分線形数理最適化だと難しいわけ方もあると思うので）

方針：
基本目標＋基本制約　で設定することにする
カスタマイズできる場所の設計方針:
- 基本変数は１つにまとめてもらう
- どうせ足すので、メンバーやチームのパラメーターは先に好きなように加工して変数１つにしてもらう

todo やりたいわけかたたち

基本目標:
- チーム全体のパラメータを設定した際にのそのパラメータのチーム間分散最小
: 平等なチーム
方針>
カスタマイズする場所> 共通 + チームメンバーのパラメーターの配列からチームパラメーター(※スカラー)への関数を設定してもらう

- チーム内相互希望度総和の全チーム総和最大化
: 相思相愛チーム優先
- チーム内相互希望度総和のチーム間分散最小
: みんなの希望を平等に聞く
方針>
  もしかしたらグラフアルゴリズムのほうがいいかも
カスタマイズする場所> 共通 + 
  
- チーム内パラメーター分散の全チーム総和最小
: 同じくらいのパラメーター　誰かに負担が偏らず仲よく

基本制約:
- チームサイズ範囲
- チームの最大パラメータ値の最小値
: 何かの話題には詳しい人を分散
- チームの最小パラメータ値の最大値
: 何かの話題には詳しくない人を分散
